mac下 Hexo+Github 搭建个人博客

发表于 2016-09-11   |     |   阅读次数

不多说, 进入正题

准备工作


※首先得有个github账号, 因为该blog依赖于github, 没有账号的同学请移步 Github

※创建一个public库(私有库收费,壕请无视), 在[ Repository name ]下面填写:yourname.github.io, 这里的yourname即是你github的用户名, 注意要全部小写!!!小写!!!小写!!!

安装环境


※基本环境: 首先需要安装git/node.js, 非本文重点, 具体步骤请移步 Google , 配置好后记得检查环境是否安装正确

※这里主要说Hexo环境配置

☛ 可以在任何有足够大小且喜欢的地方新建一个自命名的文件夹(如: xxxblog,下面内容就以该文件名进行说明), 然后通过命令行cd进入该目录下,
输入以下命令:

npm install -g hexo-cli

回车等待安装完成即可, 根据网络状况安装过程持续时间可能不同

☛ 完成后输入 hexo -v 检查是否安装正确, 如下


☛ 然后输入 hexo init 进行初始化操作, 过程会自行处理, 等待完成即可

☛ 初始化完成后再输入 npm install , 该过程会安装一些必要组件, 等待完成即可

☛ 以上操作正确后, 接下来可以体验一下本地blog功能了!!!

首先输入 hexo g (g是generate缩写)

完成后再输入 hexo s (s是server缩写)

打开浏览器(个人用的Google浏览器)输入 localhost:4000, 是不是有惊喜出现? 有就表示成功了!! 不过这个惊喜页面用的是默认主题, 下面来修改一下 _config.yml 文件内容进行一些自定义配置

发布到github


※前两步完成后你会发现 xxxblog 目录下有一个 _config.yml 文件, 可以用文本格式打开(个人用的 Sublime Text )

※然后不要犹豫, 一路向下, 滚动到最底部!! 找到下面这段:

然后修改如下:

☛ 特别提醒: 个人遇到的坑, 这里repo里面的yourname, 一定要全部小写!!!小写!!!小写!!!
lz的github账号github给的是首字母大写的用户名, 如果你也是的话, 直接改成相应的小写字母即可!!原因下面会说;
同时注意保持最少空一格, 我用的 Sublime Text , 打开会发现默认的如deploy/type会变色, 不空格的话, 不会变色, 注意语法格式问题

※下面把本地内容发布到github上

☛ 首先输入 npm install hexo-deployer-git --save

根据网络状况, 网络不好可能等待比较长的时间;

还有就是上面说的’小写’坑, 个人开始用的github默认给的大写用户名, 结果就是无限期等待!!(连个错误提示你都等不到!我是没等到!)

※然后输入 hexo d (d是deploy的缩写)

然后会要求输入你的github的账号和密码, 输入即可

这时会看到发布的一些相关输出信息, 别激动! 等待完成

☛ 注意: 上面的输入账号和密码, 如果出现错误, 这可能是之前git配置过其它账号, 但是不是你的这个github账号,
所以你需要重新修改一下配置信息, 如下两步:

git config --global user.name "xxxxx"

git config --global user.email "xxxxx@example.com"

※如果上面都没错, 这时你应该可以访问这个网址了 https://yourname.github.io, 打开即是你刚才上传的页面!!!
现在开始你也有自己的blog了!!!

※发布一篇新文章

☛ 输入 hexo n "我有blog了"

这时在 xxxblog/source/_post目录下会生成一个名为 “我有blog了.md” 的文件, 用你喜欢的编辑器打开, 开始你的blog之路吧!

完成后依次输入 hexo g, hexo s(这时可以本地查看效果), hexo d, 大功告成!!!

☛ 如果预览发现效果没有改变, 请输入 hexo clean 清空缓存, 再重复上一步进行发布

※提示: 所有以上的输入操作都在 命令行进入 xxxblog 目录下进行!

blog主题


※ here 有很多基于hexo的主题, 你可以选择一款尝试一下效果

☛ 比如 你选择了A主题

你可以选择clone或下载该主题, 放到 xxxblog/themes 下, 里面还有一个默认主题 landscape,

然后更改 _config.yml 中的 theme 字段, 改为上面下载的主题的包名 A, 保存再运行查看效果即可, 更多该主题信息可查看该主题使用说明/文档

现在你也有自己的blog了!
